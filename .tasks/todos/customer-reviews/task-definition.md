# Allow people to add lister reviews

## Overview
When a person rents an article from a lister, we want to allow them to add a review on the service provided so other people can see that and check how good the lister is while renting the published articles

## Why
Other potential renters can check the reviews and move forward with confidence about the products they want to rent

## Assumptions / Notes
- A renter might or might not want to add a review
- Reviews can have from 0 to 5 stars being 0 bad and 5 amazing listers
- A review might have a description besides the number of stars
- Any people can see the reviews for any article

## Acceptance Criteria
- [ ] A review have always the number of stars but description is optional
- [ ] A description might have as max 500 characters
- [ ] We want moderation, each time a new review is submitted, the administrator needs to approve it before being published
- [ ] A new menu option is added for admins so they can approve or reject reviews
- [ ] An admin might either approve or reject and review
- [ ] An admin might ask the renter to update their review by sending a note to the renter so they can update their review before sending it to approval again
